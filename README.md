# Ant Geometry Explorer

Eine kleine, interaktive Lernumgebung für erste Schritte in der Geometrie. Sie läuft komplett im Browser und benötigt keine zusätzliche Installation.

## Start

Öffne einfach die Datei [`index.html`](index.html) in deinem Browser.

## Funktionen

- Setze Punkte auf einer Gitternetzfläche und lies ihre Koordinaten ab.
- Miss Strecken zwischen zwei Punkten und lass dir ihre Länge anzeigen.
- Beobachte alle erstellten Punkte und Strecken in übersichtlichen Listen.
- Nutze Anregungsfragen, um Gespräche oder eigenständiges Entdecken zu starten.

## Idee für den Unterricht

- Lasse Lernende Dreiecke mit bestimmten Eigenschaften konstruieren (z. B. gleichschenklig oder rechtwinklig).
- Miss und vergleiche Strecken, um geometrische Begriffe wie Umfang und Symmetrie einzuführen.
- Nutze das Zurücksetzen, um schnell neue Konstruktionen auszuprobieren.
