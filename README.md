# Einfache Geometrie entdecken

Diese kleine Lernumgebung zeigt, wie sich Grundfiguren der ebenen Geometrie
berechnen lassen. Lernende wählen ein Rechteck, ein rechtwinkliges Dreieck oder
einen Kreis aus und verändern Maße über Regler oder Eingabefelder. Die Formeln
werden automatisch mit den aktuellen Werten gefüllt.

## Starten

Öffne die Datei `index.html` im Browser. Alle Skripte und Styles werden lokal
geladen – ein zusätzlicher Server ist nicht nötig.

## Funktionen

- **Figur wählen:** Buttons für Rechteck, Dreieck und Kreis.
- **Maße anpassen:** Regler und Zahleneingaben halten sich gegenseitig aktuell.
- **Formeln nachvollziehen:** Zwischenschritte für Fläche und Umfang zeigen, wie
  die Werte eingesetzt werden.
- **Diagramm ansehen:** Jede Figur erscheint als farbiges Schema mit den
  aktuellen Beschriftungen.

## Ideen für den Unterricht

- Lass Lernende Schätzungen abgeben, bevor sie einen Wert verändern.
- Fordere dazu auf, gleiche Ergebnisse mit unterschiedlichen Maßen zu suchen.
- Vergleiche die Formeln der Figuren und leite Gemeinsamkeiten ab.
